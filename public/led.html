Led

Introducción

La palabra LED es el acrónimo de light-emitting diode, es decir, diodo emisor de luz. Se trata de un componente electrónico pasivo cuya función es emitir luz.

Presenta varias ventajas respecto a las luces incandescentes y fluorescentes, como su bajo consumo de energía y larga vida además de corto periodo de encendido y su variedad de colores. También tiene sus desventajas, como la potencia necesaria para la iluminación de interiores.

En el siguiente ejemplo vamos a proceder a encender un led indicándole la intensidad a través de una aplicación web.

Led-Node.ino 
void setup()
{
  Serial.begin(9600);
}
void loop()
{
  while(!Serial.available());
  analogWrite(3, Serial.read());
}
